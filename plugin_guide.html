🧩 راهنمای طراحی پلاگین برای Dima CMS

پلاگین‌ها راهی برای افزودن امکانات جدید و سفارشی‌سازی سیستم بدون تغییر هسته هستند.

ساختار پلاگین

مسیرتوضیح
plugins/پوشه اصلی پلاگین‌ها
plugins/myplugin/پوشه هر پلاگین
plugins/myplugin/plugin.phpفایل اصلی پلاگین

نمونه کد پلاگین

<?php
// plugins/myplugin/plugin.php
function myplugin_init() {
    // کد شما اینجا
}
add_action('init', 'myplugin_init');

هوک‌ها و اکشن‌ها

نام هوکزمان اجرا
initهنگام بارگذاری سیستم
before_renderقبل از نمایش هر صفحه
after_renderبعد از نمایش هر صفحه

فعال‌سازی پلاگین

  1. پوشه پلاگین را در plugins/ قرار دهید.
  2. از پنل مدیریت، پلاگین را فعال کنید.
می‌توانید چندین پلاگین به صورت همزمان فعال داشته باشید.
